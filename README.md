># Crypto Exchange Bot 🤖💰

Telegram бот для обмена рублей на криптовалюту через Binance API.

## ✨ Функциональность

- 📊 Показывает текущие курсы криптовалют в реальном времени
- 💱 Обменивает рубли на крипто с расчётом комиссии
- 📈 История всех ваших транзакций
- 👤 Профиль пользователя
- 🔔 Удобный интерфейс с кнопками
- 📱 Полная поддержка Telegram

## 🚀 Установка

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/qeqeqqee/crypto-exchange-bot.git
cd crypto-exchange-bot
```

### 2. Установите зависимости
```bash
pip install -r requirements.txt
```

### 3. Получите необходимые ключи

#### Telegram Bot Token:
1. Напишите @BotFather в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен

#### Binance API Keys:
1. Зарегистрируйтесь на https://www.binance.com
2. Перейдите в "Account" → "API Management"
3. Создайте новый API ключ
4. Скопируйте API Key и Secret Key

### 4. Настройте переменные окружения

Создайте файл `.env` (используйте `.env.example` как шаблон):

```env
TELEGRAM_BOT_TOKEN=your_bot_token_here
BINANCE_API_KEY=your_binance_api_key_here
BINANCE_SECRET_KEY=your_binance_secret_key_here
ADMIN_ID=your_telegram_id_here
```

## 🎯 Запуск

```bash
python main.py
```

Бот начнет работу и будет готов к приёму команд!

## 📖 Использование

### Команды
- `/start` - Начало работы, показ главного меню
- `/help` - Справка по использованию
- `💱 Обменять` - Начать обмен рублей на крипто
- `📊 Курсы` - Показать текущие курсы
- `📈 История` - Ваши последние транзакции
- `👤 Профиль` - Информация о вашем аккаунте

### Процесс обмена

1. Нажмите "💱 Обменять"
2. Введите сумму в рублях (минимум 100 РУБ)
3. Выберите криптовалюту
4. Проверьте расчёт с комиссией
5. Подтвердите обмен
6. Следуйте инструкциям для платежа

## ⚙️ Конфигурация

### Поддерживаемые криптовалюты
По умолчанию поддерживаются:
- BTC (Bitcoin)
- ETH (Ethereum)
- BNB (Binance Coin)
- XRP (Ripple)
- ADA (Cardano)

Вы можете добавить больше в `config.py`:

```python
SUPPORTED_PAIRS = {
    "BTC": "BTCUSDT",
    "ETH": "ETHUSDT",
    "DOGE": "DOGEUSDT",  # Новая пара
    # ...
}
```

### Комиссия
По умолчанию комиссия составляет **1.5%**

Измените в `config.py`:
```python
EXCHANGE_COMMISSION = 2.0  # 2%
```

## 🗄️ База данных

Бот использует **SQLite** для хранения:
- Информации о пользователях
- Истории транзакций

База данных создаётся автоматически при первом запуске.

## 🔒 Безопасность

⚠️ **ВАЖНО:**
- Никогда не коммитьте `.env` файл с реальными ключами
- Используйте API ключи с ограничениями прав на Binance
- Не делитесь вашим Bot Token
- Храните Secret Key в безопасности

## 📊 Тестирование

Для тестирования используйте testnet Binance:
1. Создайте аккаунт на https://testnet.binance.vision
2. Измените эндпоинт в `binance_api.py`

```python
client = Client(
    BINANCE_API_KEY, 
    BINANCE_SECRET_KEY,
    testnet=True  # Добавьте эту строку
)
```

## 🐛 Решение проблем

### Ошибка "Telegram Bot Token not found"
- Проверьте, что `.env` файл находится в корне проекта
- Убедитесь, что токен правильно скопирован

### Ошибка при подключении к Binance
- Проверьте API ключи
- Убедитесь, что IP адрес в списке разрешённых на Binance
- Проверьте интернет соединение

### Бот не отвечает
- Проверьте, что `main.py` запущен без ошибок
- Посмотрите логи в консоли
